## Rakeの基本的な使い方

- Rakefile
  ```ruby
  namespace :my_tasks do   # 名前空間
    desc 'テスト用タスク'  # 説明
    task :hello do         # タスク hello を定義
      puts 'Hello, world!'
    end
  end
  ```

- タスク実行
  ```bash
  $ rake my_tasks:hello
  # => Hello, world!
  ```

- タスク一覧
  ```bash
  $ rake -T
  # => rake my_tasks:hello  # テスト用タスク
  ```
